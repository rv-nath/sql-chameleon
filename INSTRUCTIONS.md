# Instructions for the AI tool

## Project instructions

- Build the program using rust language.
- support command line options for
  -- source file (required)
  -- source dialect (optional - default MySQL assumed)
  -- target file (optional- console output)
  -- target dialect (optional - MySQL)



## Generic instructions
- Do things one at a time. Don't rush to create all the code at the jump of the start.
- Create a skeleton
- Fill the things one by one. 
- The idea is to let the user working with the AI tool to become fully knowledgeable and hence needs handholding.  So do NOT rush.
